# Maori-Hyperledger
Maori Hyperledger dev

This project aims to develop a Hyperledger solution to help the Māori government to register various items:
- Land / property
- Members
- Whānau
- Hapu
- Businesses
- Trusts
- etc...

This is the dev environment that will consist of the following:
- Waka-district1 peer
- ca-Waka-district1
- Waka-district2 peer
- ca-Waka-district2
- Maori-orderer
